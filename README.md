# Supermarket Sales Insights

## Overview
This Excel dashboard analyzes supermarket sales data to evaluate revenue trends across branches, product lines, customer types, and time periods.

## Tools Used
- Excel Pivot Tables
- Pivot Charts
- Basic Data Cleaning

## Key Insights
- Identified the highest-performing branch by total revenue
- Determined peak sales hours and busiest days
- Analyzed top-performing product lines
- Compared customer type performance across branches
